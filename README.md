#  Face Recognition Based Attendance System

A Python-based Face Recognition Attendance System that uses OpenCV and computer vision to automatically detect faces 
and mark attendance in real-time.

##  Features

-  Real-time face detection using webcam  
-  Face recognition using trained dataset  
-  Automatic attendance marking  
-  Attendance stored in CSV file  
-  Simple and user-friendly implementation
